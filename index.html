GitHub 文件加速

GitHub 文件链接带不带协议头都可以，支持 release、archive 以及文件，右键复制出来的链接都是符合标准的

release、archive 使用 cf 加速，文件会跳转至 JsDelivr

注意，不支持项目文件夹

例如1：https://github.com/ac/project/archive/master.zip

例如2：https://github.com/ac/project/archive/v0.1.0.tar.gz

例如3：https://github.com/ac/project/releases/download/v0.1.0/example.zip

例如4：https://github.com/ac/project/blob/master/filename
